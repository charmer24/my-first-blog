# Django Blog

A personal blog website built with Django and Python.

## About

This project is a full-stack blog application where users can view blog posts and content can be managed through Django's admin panel.

## Features

- Blog post listing and detail pages
- Django admin dashboard for managing posts
- Custom templates and styling
- SQLite database
- Static file management
- Deployed using PythonAnywhere

## Built With

- Python
- Django
- HTML
- CSS
- SQLite

## Project Structure

- `blog/` - Blog application containing models, views, forms, and URLs
- `mysite/` - Django project settings and configuration
- `templates/` - HTML templates
- `static/` - CSS and static files
